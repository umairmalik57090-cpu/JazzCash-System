# JazzCash-System
Programming Fundamentals Project in C++
This is a Programming Fundamentals project developed in C++.  
It simulates a basic JazzCash system with user registration, login, money transfer, and bill payment features.

## 🚀 Features
- User Registration & Login System
- Send Money (JazzCash, Easypaisa, Bank)
- Bill Payment System (Water, Gas, Electricity)
- Input Validation (numbers, length checks)

## 🛠️ Technologies Used
- C++
- Console-based application
